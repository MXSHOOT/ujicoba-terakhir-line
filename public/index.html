jika ingin running the app using Node.js, tolong set LIFF ID terhadap environment variable di account Heroku anda lalu ikuti step by stepnya:

  1. Buka `Dashboard` di akun Heroku .
  2. klik app lalu klik created.
  3. klik `Settings` dan cari `Reveal Config Vars`.
  4. masukkan `MY_LIFF_ID` untuk key nya dan value atau nilai, masukkan LIFF ID (cth : 165X856817-X8GXXVNr ada di url LIFF).
  5. kalau sudah, coba refresh kembali web nya.

jika belum bergabung, tolong gabung LIFF ID di sini index.html file.
